• What is HTTP?

    • HTTP is an abrievation for HyperText Transfer Protocol. It is a protocol used to send or recieve data from a server.

  • What is a URL?

    • URL is an abrievation for Uniform Resource Locator. It is basically the adress of a web page.

  • What is DNS?

    • DNS is an abrievation for Domain Name System which translates the human readable domain name into an ip address for the system.

  • What is a query string?

    • A query string is additional data sent to a a web page.

  • What are two HTTP verbs and how are they different?

    • HTTP verbs are methods in which http uses to request data. Two examples are push and get. Get simply retrieves data as is, while push can alter and update data that it is requesting.

  • What is an HTTP request?

    • An HTTP request is a request from a client to a server to retrieve data.

  • What is an HTTP response?

    • An HTTP response is the response given after a request is sent to the server from teh client.

  • What is an HTTP header? Give a couple examples of request and response headers you have seen.

    • HTTP headers provide additional information about a request or a response.

  • What are the processes that happen when you type “http://somesite.com/some/page.html” into a browser?

    • The browser runs the http protocol and the dns then turns the site name into an ip address for the system to request data from. It then passes in the information "some/page.html" to give you a more accurate place within the server to load into.
